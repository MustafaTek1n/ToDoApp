📝 Console-Based To-Do App (Java)
Hello! This is a task management tool I developed during my software engineering studies to put core principles like OOP, Data Management, and Error Handling into practice.

Beyond just writing code, I focused on creating a logical flow that genuinely considers the user's experience within a terminal environment.

✨ Key Features
Smart Sorting: The app knows what's urgent. It automatically organizes tasks based on completion status, priority levels (High to Low), and due dates.

Persistent Data: Your tasks won't vanish when you close the app. Using Java Serialization, I implemented a system that saves everything to a tasks.dat file and reloads it instantly upon startup.

Robust Input Validation: The app is designed to be "crash-proof." If you enter text where a date is expected, it won't break; instead, it provides a helpful prompt to guide you to the correct format.

Quick Search: Even with dozens of tasks, you can find exactly what you're looking for with a simple keyword search.

🛠 Technical Deep Dive
I focused on utilizing the Java Standard Library efficiently to ensure the app is lightweight yet powerful:

Java 8+ Time API: Precise date management using the YYYY-MM-DD format for a disciplined schedule.

Object Streams: Direct object-to-disk persistence (Serialization).

Encapsulation: Ensuring data integrity and clean communication between class components.

🚀 How to Run
If you have Java installed, you can try it out with these two commands:

Bash
javac ToDoApp.java
java ToDoApp
